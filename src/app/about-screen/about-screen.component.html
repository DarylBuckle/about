About This Site


My Portfolio page is built using a template I wrote in Angular called angular-dev-portfolio.

I made it initially as a way of learning Angular. The idea is all the data is structured in objects and the data resides inside a single file, "data.ts", so it is very easy to use the repository as a template and in a matter of minutes put your own data into the template. It uses Bootstrap so the theme can easily be changed in SCSS. The site has no back-end and is hosted on GitHub pages (it's free!).

This template is free to use. See my blog post on setting up your own portfolio page, located here.
